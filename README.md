# Hospital Management System

## Workflow
- Admin logs into the system securely.
- Admin manages doctors, patients, rooms, and appointments.
- Doctors add and manage their available timing schedules.
- Patients book appointments based on doctor availability.
- The system stores hospital records and appointment details in the database.
- Admin can create, update, view, and delete records anytime.
- Authentication and authorization provide secure access control.

## Technologies Used
- Frontend: HTML, CSS, Bootstrap, Razor Views
- Backend: ASP.NET Core MVC (.NET 8), C#
- Database: SQL Server
- ORM: Entity Framework Core
- Authentication: ASP.NET Core Identity
- Architecture: MVC Architecture
- IDE: Visual Studio 2022
- Version Control: Git & GitHub
